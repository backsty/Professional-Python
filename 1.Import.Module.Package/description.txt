1. Пакет в Python:
Пакет в Python - это специальная структура организации модулей. Пакет представляет собой директорию, которая содержит
файл с именем __init__.py и, возможно, другие поддиректории и файлы. Используя пакеты, вы можете организовать и
структурировать ваши модули логически.

Например, если у вас есть директория "mypackage", содержащая файлы "module1.py" и "module2.py", чтобы превратить эту
директорию в пакет, необходимо добавить файл __init__.py внутрь этой директории. Тогда "mypackage" станет пакетом,
содержащим модули "module1" и "module2".

2. Директория:
Директория в общем смысле - это просто место для хранения файлов. В контексте Python, директория может быть просто
коллекцией файлов, включая модули Python, а также другие директории. Директория сама по себе не имеет специального
значения в Python, если в ней нет файлов с определенными именами, например, __init__.py для создания пакетов.

Таким образом, пакет является специальным типом директории в Python, который содержит определенные файлы и директории,
организованные специальным способом для использования в качестве модулей или подпакетов в вашем приложении.
